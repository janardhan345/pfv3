Chess's True Purpose

July 08, 2026

Chess is considered to make our minds sharp and many call it as the games for Genius, but I see something Odd when current players play it cause its more of memorizing the methods to defeat the opponent or practicing to checkmate with patterns, but chess's true purpose is to analyze the moves you have right now and deciding the best move cause every move will affect you in long term and you have to select the suitable move within the limited time. Its for decision making under pressure.

Similarly DSA became the very thing like Chess losing it's true purpose due to competition and interviews. DSA isn't for competitive programming, it is used to built many things that we use in our day to day life, it's the foundation for implementing many features in OS such as file management, process scheduling etc. these are used everywhere for example do/undo in code editor, forward/ backward in browser, version control in git etc. The Next Time you see an Data structure and Algorithms problem remember how it's implemented in the real world and what purpose it serves or even try to replicate them for a good practice.
